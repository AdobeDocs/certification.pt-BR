# Jornada de certificação para Adobe [!DNL Experience Manager Sites] Especialista em profissionais

Aprenda a se tornar um especialista Adobe.

## Experiência mínima

Você deve ter:

* Experimente os recursos e as AEM necessárias para envolver os desenvolvedores a encontrar soluções de negócios
* A capacidade de usar AEM produtos para gerenciar conteúdo para impulsionar experiências digitais
* Uma compreensão de quais módulos estão presentes
* Um insight mais profundo em módulos como [!DNL Sites], [!DNL Assets]e [!DNL Forms]

Além disso, você deve conhecer as seguintes tecnologias e ambientes:

* Ferramentas de gestão de conteúdo
* Ferramentas de gerenciamento de ativos digitais
* Projetos
* Forms
* Fragmentos de experiência e fragmentos de conteúdo
* Screens
* Fluxos de trabalhos
* Inicializações
* CRXDE Lite
* Administração do usuário
* Gerenciamento de vários sites
* Ferramentas de monitoramento
* Controle de versão
* Hub de personalização e contexto
* Editor SPA
* Marcação com tags
* Uso do navegador
* Compreensão geral da Web
* Conceitos responsivos de design da Web

>[!NOTE]
>
>Sua certificação é válida por dois anos a partir do momento em que for aprovada. Recomendamos ter a versão mais recente quando possível, pois ela abrange recursos de produto mais recentes.

**Objetivos e âmbito do exame**

Seção 1: Educação (22%)

* Listar recursos e recursos AEM relevantes
* Recomendar como aproveitar AEM recursos para atender às necessidades dos negócios
* Aplique os conceitos de procedimento necessários para realizar o treinamento de editores de conteúdo

Seção 2: Arquitetura (24%)

* Recomendar uma arquitetura de informações para aproveitar AEM recursos padrão para atender às necessidades dos negócios usando o ambiente no local ou Adobe hosted (Cloud)
* Descreva como o AEM se integra a serviços de terceiros
* Determinar como definir funções e permissões
* Descrever abordagens de armazenamento em cache

Seção 3: Análise de negócios (38%)

* Mapear módulos padrão de AEM para diferentes problemas de negócios para o cliente
* Aplicar conceitos de gerenciamento de conteúdo para módulos de AEM
* Identificar os participantes típicos do projeto
* Aplicar conceitos processuais necessários para apoiar a colaboração de todas as partes envolvidas
* Determine como cumprir as principais metas de negócios, mantendo-se dentro do orçamento e/ou escopo
* Determine como as perguntas dos desenvolvedores devem ser respondidas em relação às metas comerciais

Seção 4: Configuração e implementação (16%)

* Determine como configurar e usar componentes de conteúdo para mostrar as práticas recomendadas do AEM
* Verifique a implementação correta dos recursos e localize e descreva os erros que ocorrem

**Fazer um teste de prática**

Recomendamos fazer o teste de prática livre antes de começar. Você terá acesso irrestrito a todos os testes de prática disponíveis para exames no nível do especialista Adobe.

Nossos testes de prática são desenvolvidos a partir do mesmo blueprint dos exames ao vivo. O teste de prática ajudará você a entender os tópicos abordados no exame ao vivo.

Lembre-se de que passar no teste de prática não garante que você passará no exame de certificação.

Vá para o [Portal de preparação de teste de certificação Adobe](https://www.certmetrics.com/adobe/candidate/gmetrix_sso.aspx){target="_blank"} agora para fazer um teste de prática.

**Faça o exame de certificação**

O Adobe usa dois fornecedores para administrar exames de produtos, PSI e Examity.

* **PSI** oferece opções de teste de centro e teste japonês (quando disponível)
* **[!DNL Examity]** oferece uma opção online remota

Para agendar, reprogramar ou cancelar seu exame, basta clicar no fornecedor escolhido e seguir as instruções fornecidas na tela. É tão simples assim!
