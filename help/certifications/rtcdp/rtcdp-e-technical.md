# Jornada de certificação para Adobe [!DNL Real-Time Customer Data Platform] Expert

>[!IMPORTANT]
>
>Atualmente, a certificação de Adobe [!DNL Experience Platform] Expert requer a aprovação em dois exames: **Adobe [!DNL Experience Platform] Technical Foundations (AD0-E600)** e **Adobe [!DNL Real-Time CDP] Technical Practitioner (AD7-E601)**. A partir de 15 de janeiro de 2024, um novo exame será iniciado, e esse será apenas o exame necessário para essa certificação.

### Público-alvo {#intended-audience}

* Arquiteto/engenheiro de dados
* Arquiteto de soluções ou MSA
* Arquiteto corporativo
* Gerente de produto
* Líder de tecnologia de marketing
* Consultor técnico

<br>

### Experiência mínima {#minimum-experience}

Você deve ter:

* Conhecimento básico do [!DNL Experience Platform] e sua navegação pela interface do usuário
* A capacidade de criar segmentos e públicos-alvo (lógica incluída nele)
* A capacidade de configurar destinos
* A capacidade de conectar segmentos a destinos
* A capacidade de criar insights/eventos para responder e reagir em tempo real
* A compreensão da configuração do DULE durante a assimilação de dados, limitação de segmentos durante as ativações
* A compreensão dos requisitos de aceitação por canal
* A capacidade de projetar/entender os dados que precisam ser capturados e ativados após o uso das práticas recomendadas para a frequência e o volume dos dados do caso de uso
* A capacidade de compreender os dados necessários em diferentes canais
* A capacidade de determinar quais atributos de perfil são necessários
* Conhecimento da frequência de envio de segmentos por meio de destinos
* Conhecimento da frequência com que a associação do segmento é atualizada

## Requisito #1: Informações para Adobe [!DNL Experience Platform] Technical Foundations (AD0-E600) {#req-1}

>[!IMPORTANT]
>
>A programação para o exame AD0-E600 foi colocada em espera. Este exame está sendo atualizado e será lançado em **15 de janeiro de 2024**. Após essa data, essa certificação exigirá apenas a aprovação em um exame, em vez dos dois exames atualmente necessários.
>
><br>
>
>**Se você já tiver agendado o exame AD0-E600**, recomendamos cancelar seu compromisso no Portal de Exame (veja a etapa 3 abaixo) e reagendar quando o novo exame for lançado. Se tiver dúvidas, contate o [Suporte à Certificação](mailto:certif@adobe.com).

O exame AD0-E600 valida as habilidades e conhecimentos que abrangem os recursos básicos e as tarefas realizadas com frequência do Adobe [!DNL Experience Platform] no nível Expert.

### Detalhes do exame para AD0-E600 {#exam-details-600}

* Nível: especialista (1-3 anos de experiência)
* Pontuação de aprovação: 35/50
* Tempo: 105 minutos
* Entrega: on-line vigiado (requer acesso à câmera) ou centro de teste vigiado
* Idioma: inglês
* Custo: US$ 225 (global) / US$ 150 (Índia)
* Identificação do Exame: AD0-E600

<br>

### Objetivos do exame e escopo para AD0-E600 {#exam-objectives}

{{exam-objectives}}

**Seção 1: Modelagem de Dados (20%)**

* Analisar dados de origem para avaliar as identidades primária e secundária para a compilação de perfis
* Demonstrar uma compreensão de como usar a interface do para criar/editar esquemas XDM.
* Identificar a orientação do DULE

**Seção 2: Assimilação de dados (36%)**

* Demonstrar como formatar e preparar dados para assimilação
* Demonstrar como conectar fontes de dados usando conectores OOTB
* Demonstrar como assimilar dados de origem por meio de lote e transmissão
* Descrever como monitorar as transferências de dados
* Demonstrar como executar a detecção de dados nos dados de origem
* Demonstrar como transformar dados para corresponder ao XDM

**Seção 3: Unificação de Perfis (34%)**

* Validar perfis e dados do evento após a assimilação
* Definir espaços de nome de identidade
* Explicar como os gráficos de identidade são usados pelo serviço de perfil
* Identificar como enriquecer perfis por meio de serviços de modelagem de espaço de trabalho de ciência de dados
* Demonstrar como criar um segmento com o Construtor de segmentos e como o serviço de segmentação funciona

**Seção 4: Ativação (10%)**

* Demonstrar como configurar um destino e como a ativação de segmentos funciona
* Demonstrar e entender a API de acesso a dados e exportar dados por meio do Real-Time CDP

## Requisito #2: Informações para o profissional técnico do Adobe [!DNL Real-Time CDP] (AD7-E601) {#req-2}

>[!IMPORTANT]
>
>Este exame está sendo atualizado e será lançado em **15 de janeiro de 2024**. Após essa data, essa certificação exigirá apenas a aprovação em um exame, em vez dos dois exames atualmente necessários.
>
><br>
>
>**Se você não tiver feito nenhum dos exames necessários no momento,** recomendamos que aguarde o exame atualizado. Dessa forma, você pode fazer um exame em vez de dois.
>
><br>
>
>**Se você já tiver passado no exame AD0-E600 Adobe [!DNL Experience Platform] Technical Foundations e estiver se preparando para o exame AD7-E601**, sugerimos que você mantenha sua consulta, pois a aprovação no exame AD7-E601 concederá a Certificação de Especialista do Adobe [!DNL Real-Time Customer Data Platform]. Se tiver dúvidas, contate o [Suporte à Certificação](mailto:certif@adobe.com).

O exame AD7-E601 abrange a criação, o gerenciamento e a compreensão do perfil do cliente, a segmentação, os destinos entre fontes e outros tópicos. O exame valida as habilidades e o conhecimento fundamentais necessários para empregar a Real-time CDP para segmentação de público, exportações de destinos e ativação em tempo real para perfis unificados que seguem as regulamentações de dados e privacidade. Os candidatos devem ter experiência com CDP (Plataforma de dados do cliente) e conhecimento sobre o Adobe [!DNL Experience Platform].

<br>

### Detalhes do exame para AD7-E601

* Nível: especialista (1-3 anos de experiência)
* Pontuação de aprovação: 34/25
* Tempo: 51 minutos
* Entrega: on-line vigiado (requer acesso à câmera) ou centro de teste vigiado
* Idioma: inglês
* Custo: US$ 225 (global) / US$ 150 (Índia)
* ID do exame: AD7-E601

<br>

### Objetivos e âmbito do exame para AD7-E601

{{exam-objectives}}

Seção 1: Validação(41 %)

* Aplique os conceitos necessários para validar se o Perfil tem os atributos e segmentos adequados
* Demonstrar uma compreensão de como validar uma estratégia de implantação com base em requisitos funcionais ou técnicos
* Analisar variáveis relevantes para implementar o Fluxo End to End (E2E) da RTCDP
* Monitorar exportação de segmento para destino
* Aplicar conceitos necessários para validar políticas de governança de dados

Seção 2: Segmentação (32%)

* Demonstrar uma compreensão de como criar segmentos com base nas necessidades dos negócios
* Demonstrar uma compreensão de como determinar avaliações de segmento com base nos dados assimilados

Seção 3: Destinos (27 %)

* Aplicar os conceitos necessários para configurar um ou mais Destinos ao enviar/exportar públicos com base em casos de uso de negócios
* Demonstrar compreensão da exportação de segmentos por meio do Destino

<br>
